Reinforcement learning as three overlapping questions

Does it use a model · does it learn values · does it learn an explicit policy. The overlaps are the hybrids.

Model-based — plans with dynamics P, R
Value-based — greedy on learned values
Policy-based — optimises π directly
Value ∩ Policy = actor-critic
All three = AlphaZero, MuZero
On/off-policy is a separate axis — use the filters
