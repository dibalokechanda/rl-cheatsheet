# Reinforcement Learning Taxonomy & Visuals

This project is a unified interactive explorer of reinforcement-learning concepts, algorithms, notation, and equations.

## Visualization

Open [index.html](index.html) in a modern browser to explore:

- A navigable RL taxonomy
- Equation and notation references
- An infinite canvas
- A Venn diagram of RL algorithm families

There are no build steps or heavy dependencies required.

## Usage

```bash
git clone https://github.com/dibalokechanda/rl-cheatsheet.git
cd rl-cheatsheet
```

Then open `index.html` in a modern browser.
