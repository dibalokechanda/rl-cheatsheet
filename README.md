# Reinforcement Learning Taxonomy & Visuals

Welcome to the **RL-Taxonomy-Visuals** repository! This project contains a collection of interactive HTML visualizations and cheat sheets designed to help you explore and understand the vast landscape of Reinforcement Learning (RL) concepts, algorithms, and equations.

## 🚀 Visualizations

You can view these interactive tools directly by downloading the HTML files and opening them in your browser, or via the GitHub Pages links if enabled:

- **[RL Explorer](index.html)**
  An interactive explorer for RL concepts, providing a navigable graph or tree of how different RL algorithms and ideas are connected.

- **[RL Equations](rl-equations-tabs.html)**
  A tabbed interface detailing the core mathematical equations governing various RL algorithms (e.g., Bellman equations, Policy Gradients).

- **[RL Infinite Canvas](rl-infinite-canvas.html)**
  A draggable, zoomable infinite canvas mapping out the taxonomy and relationships within Reinforcement Learning.

- **[RL Venn Diagram](rl-venn.html)**
  A visual Venn diagram illustrating the intersections of different subfields within RL (e.g., Model-Free vs. Model-Based, On-Policy vs. Off-Policy).

## 🛠️ Usage

These tools are built as standalone, single-file HTML pages. There are no build steps or heavy dependencies required. 
Simply clone the repository or download the HTML files, and open them in any modern web browser!

```bash
git clone https://github.com/dibalokechanda/rl-cheatsheet.git
cd rl-cheatsheet
```
Then, just double-click any `.html` file.

## 🌐 Publishing to GitHub Pages

To make these visualizations available online as a live website:
1. Go to your repository on GitHub.
2. Click on the **Settings** tab.
3. On the left sidebar, click on **Pages**.
4. Under the **Build and deployment** section, select `Deploy from a branch`.
5. For the **Branch**, select `main` and the `/ (root)` folder, then click **Save**.
6. Wait a minute or two, and your site will be live! 

*(Since this README serves as the main page, the links above will automatically work and route visitors to your different tools!)*
